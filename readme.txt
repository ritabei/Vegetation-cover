The data file "ESA_landcover_data.csv" contains the vegetation types of the ESA CCI LC 
land cover classification scheme, as well as the latitude (lat), longitude (lon) coordinates,
and elevation of each observation.

columns 1-10: vegetation type
column 11: longitude
column 12: latitude
column 13: elevation

Vegetation types:
e1: Tree_Broadleaf_Evergreen 
e2: Tree_Broadleaf_Deciduous
e3: Tree_Needleleaf_Evergreen
e4: Tree_Needleleaf_Deciduous
e5: Shrub_Broadleaf_Evergreen
e6: Shrub_Broadleaf_Deciduous
e7: Shrub_Needleleaf_Evergreen
e9: Natural_Grass
e11: Bare_Soil
e13: Snow_Ice
